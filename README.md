## 简介

**WinUI3 Media Viewer** 是一款基于 WinUI 3 的轻量级媒体查看器，支持拖拽浏览图片（JPG/PNG/ICO）、音频（MP3）和视频（MP4）。界面简洁，操作直观，适合日常快速预览媒体文件。

## 特性

- 📁 **拖放加载** – 将文件或文件夹拖入窗口即自动识别并加载所有支持的媒体。
- 🖼️ **图片预览** – 支持 JPG、PNG、ICO 等格式，自动生成缩略图列表。
- 🎵 **音频播放** – 内置播放控制器，支持 MP3 及常见音频格式。
- 🎬 **视频播放** – 基于系统 MediaPlayer，流畅播放 MP4 等视频。
- 🔍 **快速切换** – 单击左侧列表项即可切换预览内容。
- 🧹 **一键清空** – 随时清空当前所有文件。

## 系统要求

- Windows 10 版本 1809（内部版本 17763）或更高版本
- 已安装 [Windows App SDK 运行时](https://learn.microsoft.com/zh-cn/windows/apps/windows-app-sdk/downloads)

## 开发环境

- Visual Studio 2022（17.0 或更高）
- .NET 6.0 SDK
- Windows App SDK 1.5
